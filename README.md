# Hak5
Scripts for Hak5 tools, made by KN Ping members

# Tools available
KN Ping owns following Hak5 tools:
- Rubber Ducky
- Bash Bunny
- Key Croc
- Shark Jack Cable
- Screen Crab
- Packet Squirrel mark 1
- O. MG Cable Basic

# File structure
    .
    ├── Rubber Ducky                    # Rubber Ducky scripts
    │   ├── Member's Name               # Scripts made by said member
    │   │   ├─  Readme.md               # Quick overview of included scripts & demonstration instructions
    │   │   ├─  Demonstration           # Files needed for practical demonstration
    │   │   └─  Scripts
    │   └── ...
    └─ ...                              # Scripts for other tools

# Contributing
To contribute simply create new branch and follow our file structure.
When you're done, create Pull Request and request a review from one of the Ping's Higher-ups
